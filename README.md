# Python - Setup Project

A flexible, organized, and clean Python 3.10 project structure ready for implementing new projects with Conda.

## Conda Snippets

### Create environment

``` conda env create -n talker-env -f ./env.yml ```

### Update environment

``` conda env update -n talker-env -f ./env.yml ```

### Remove environment

``` conda env remove --n talker-env ```

### Activate environment

``` conda activate talker-env ```

### Deactivate environment

``` conda deactivate ```

## Run Project

``` python src/main.py ```

## Notebooks

``` notebooks/main.ipynb ```
